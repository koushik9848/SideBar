About Me

Hi, I’m Koushik, a student from Telangana, India, learning web development. I enjoy working with technologies like HTML, CSS, and JavaScript to build websites and simple applications. So far, I’ve created projects like a Dice Game, Timer, Stopwatch, and Calculator, which have helped me improve my coding skills.


Projects

I have worked on several projects to develop my coding skills. Here's a brief description of each one:

  • Dice Game: This fun dice rolling game allows two players to roll their dice by clicking the "Roll" button. The game generates two random numbers (one for each player) between 1 and 6. The higher number wins, and the result is displayed as either "Player 1 Wins," "Player 2 Wins," or "It's a Draw" if both players roll the same number. The numbers are generated using Math.random(), and the results are shown dynamically on the screen.

  • Timer: This countdown timer is set to December 26, 2024, the date of your semester exams. You can start the timer, and it will count down the days, hours, minutes, and seconds until that date. The timer is updated in real-time and displays a message when the countdown reaches zero. This helps keep track of the time remaining for your exams. It uses JavaScript's setInterval() function to update every second, and new Date() to calculate the difference between the current time and the target date.

  • Stopwatch: A simple stopwatch that counts up from zero when started. The user can pause, resume, or reset the stopwatch. It uses setInterval() to increment the time every second, and clearInterval() to stop or reset the timing.

  • Calculator: A basic calculator for performing arithmetic operations (addition, subtraction, multiplication, division). The user inputs numbers and selects an operation, and the result is displayed. The logic for the calculations is handled by JavaScript operators, with error handling for invalid input like division by zero.
